# sensAI Studio
This GitHub organization hosts multiple repositories that provide all the essential tools, code, and sample models for training neural networks and performing inference on supported Lattice FPGAs.

# sensAI Studio Repositories
| Name | Access | Content Overview |
| --- | --- |  --- |
| [sensAI Tools](https://github.com/sensAI-Solution/sensAITools) | Private | Tools needed to train and quantize models for Lattice FPFAs.  This also incudes simulation and analysis tools. You must [submit a request](request_access.md) to access this content. |
| [sensAI Compiler](https://github.com/sensAI-Solution/sensAICompiler) | Public | The compiler supports generating a Lattice FPGA runtime from a trained neural network models. Supports Windows and Linux.  |
| [sensAI Reference Designs](https://github.com/sensAI-Solution/sensAIReferenceDesign) | Public | Contains various human machine interface, object detection and defect detection reference desings.  |
| [sensAI Model Zoo](https://github.com/sensAI-Solution/sensAIModelZoo-Public) | Public | Provides sample models as well as previews of models designed for Lattice FPGAs. |
| [sensAI Model Zoo Private](https://github.com/sensAI-Solution/sensAIModelZoo-Private) | Private | Multiple FPGA compatible models and training scripts. You must [submit a request](request_access.md) to access this content. |
